# 📄 Zarinpal Website Template (HTML & CSS)


This project is a simplified redesign of the Zarinpal homepage, built entirely with pure HTML and CSS. The goal of this project is to practice UI development without using any external frameworks or libraries.

---

### 🌐 Live Demo
You can view the deployed version of the project here:  
https://real-estate-afshinsahrae.vercel.app/

---


### 🚀 Features

- Clean and minimal implementation of Zarinpal’s homepage layout  
- Built with pure HTML and CSS (no frameworks)  
- Easy-to-understand and extendable structure  
- Basic responsive design  
- Follows fundamental UI/UX principles  

---

### 🛠️ Technologies Used

| Technology | Description |
|-----------|-------------|
| HTML5     | Page structure and content |
| CSS3      | Styling and layout design |

---


### ⚙️ How to Run

1. Clone the repository:
 `
  https://github.com/afshinsahrae/zarrinpal-template.git
   `

2. Open the index.html file in your browser.
